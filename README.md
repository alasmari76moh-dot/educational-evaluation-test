# اختبار شامل في التقويم التربوي والمؤسسي والبرامجي

موقع اختبار تفاعلي عربي متجاوب بدون اتصال بالإنترنت، يحتوي على 100 سؤال متعدد الخيارات.

## الملفات الجديدة

- `index.html` — هيكل الصفحة الرئيسية
- `styles.css` — التنسيق والتصميم المتجاوب
- `app.js` — منطق الاختبار والتنقل والتصحيح
- `exam-data.js` — بيانات الأسئلة المأخوذة من `exam_100_mcq_ar.json`
- `README.md` — هذا الملف

## كيفية التشغيل

### الطريقة الأولى: فتح الملف مباشرة

افتح ملف `index.html` مباشرة في المتصفح بالنقر المزدوج عليه.

> ملاحظة: بعض المتصفحات قد تفرض قيودًا على `localStorage` عند فتح الملف مباشرة من المسار `file://`. إذا واجهتَ مشكلة في حفظ التقدم، استخدم خادمًا محليًا.

### الطريقة الثانية: خادم محلي

1. افتح الطرفية (Terminal) داخل مجلد المشروع.
2. نفّذ الأمر التالي:

```bash
python3 -m http.server 8080
```

3. افتح المتصفح على العنوان:

```
http://localhost:8080
```

## الميزات

- واجهة عربية من اليمين إلى اليسار (RTL).
- عرض سؤال واحد في كل مرة مع إمكانية التنقل.
- شريط تقدم يوضح عدد الأسئلة المجابة.
- شبكة أرقام الأسئلة للانتقال السريع.
- حفظ تلقائي للإجابات والسؤال الحالي في `localStorage`.
- حساب النتيجة والنسبة المئوية والدرجة.
- تحليل الأداء حسب المجالات.
- مراجعة كاملة للإجابات بعد التسليم مع إظهار الإجابة الصحيحة.
- لا يظهر الإجابات الصحيحة قبل تسليم الاختبار.
- متجاوب مع الجوال وسطح المكتب.
- لا يحتاج إلى مكتبات خارجية أو اتصال بالإنترنت.

## مقياس الدرجات

| النسبة المئوية | الدرجة |
|---|---|
| 90 – 100 | ممتاز |
| 80 – 89 | جيد جدًا |
| 70 – 79 | جيد |
| 60 – 69 | مقبول |
| أقل من 60 | يحتاج مراجعة مركزة |

## المجالات التحليلية

1. تعريفات التقويم وأنماطه — الأسئلة 1 إلى 20
2. المعايير والأدوات والمستويات — الأسئلة 21 إلى 30
3. التقويم المؤسسي والبرامجي والمفاهيم — الأسئلة 31 إلى 50
4. الأهداف والجودة والمعايير والمؤشرات — الأسئلة 51 إلى 70
5. مؤشرات الأداء والنماذج والتصميم و CIPP — الأسئلة 71 إلى 100
